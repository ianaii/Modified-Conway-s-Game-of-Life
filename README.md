# Modified Conway's Game of Life
A modified terminal-based implementation of Conway's Game of Life
Made By: Ian
License: MIT License

## Overview

Modified Conway's Game of Life is a **CLI** (Command-Line Interface) based 0-player game based on the original Conway's Game of Life (which is Turing Complete!), but due to the limitations of the Terminal, the grid is instead finite rather than infnite. The states of the cells are dictated by these rules; unchanged from the original Game of Life:

## Rules
from: https://en.wikipedia.org/wiki/Conway's_Game_of_Life

1. Any live cell with fewer than two live neighbors dies, as if by underpopulation.

2. Any live cell with two or three live neighbors lives on to the next generation.

3. Any live cell with more than three live neighbors dies, as if by overpopulation.

4. Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction
